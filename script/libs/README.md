# LuatOS包仓库

## 介绍

存放和管理LuatOS的库文件, 包含纯C/纯Lua/Lua-C混合的各种库文件

## 包管理要求

1. 语法正确. 请尽量兼容air和soc固件
2. 包含package.json, 添加基本的描述信息

## 如何提交新的库或修改

1. 欢迎使用pull request提交
2. 若使用git有困难, 也欢迎通过邮件方式提交到 wendal@openluat.com
3. 提交到本库的代码, 自动应用MIT协议, 请留意代码来源, 确保合法合规
